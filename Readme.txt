-----------------------------------------------------------------------------------------------------------------------------------------
Rss Feed display a.k.a. Rss Feed to Slideshow is an Assignment designed and developed for rtCamp Solutions Pvt. Ltd.

Details:
1. Fetch a RSS feed using PHP. Feed URL – http://devilsworkshop.org/feed
2. Parse titles of each post. (there are 10 titles for 10 posts)
3. Parse content part of each post to extract first image tag <img>. You can use regular expression pattern here.  4. Output 10 titles & image pairs as HTML.
5. Add jQuery-slider to rotate title+image pairs as slideshow.
-----------------------------------------------------------------------------------------------------------------------------------------
As a small updation in it I also added a AJAX loading to the slideshow.
"Galleria framework" an JavaScript image gallery framework is used to built the Slideshow. More details on galleria.io

Thanks for Galleria framework and Google.
